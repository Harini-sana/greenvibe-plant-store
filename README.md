# 🌿 GreenVibe – Plant E-Commerce Frontend

GreenVibe is a frontend e-commerce web application designed to simulate a modern online plant store.
This project was built to strengthen skills in UI/UX design, responsive layouts, and real-world frontend logic using JavaScript.

---

## 🚀 Features

* 🛍️ Product listing with categories & filters
* 🔍 Search functionality
* ❤️ Wishlist system (stored in localStorage)
* 🛒 Add to Cart with dynamic updates
* 📦 Cart management & order simulation
* 👤 User dashboard (orders, profile, settings)
* 🌟 Premium product popup interaction
* 📱 Fully responsive design
* 🎨 Clean and modern UI with animations

## 📂 Project Structure

```bash
index.html        # Home page  
shop.html         # Product listing page  
cart.html         # Cart page  
dashboard.html    # User dashboard  
contact.html      # Contact & FAQ  
wishlist.html     # Wishlist page  
```

---

## 💡 Key Functional Logic

* Cart & Wishlist stored using `localStorage`
* Dynamic rendering of products using JavaScript
* Event-based UI interactions (modals, filters, toggles)
* Navigation handled between pages with state persistence
